# Airline Reservation System (Java)

A simple console-based airline seat reservation system.

## Features
- Reserve seats
- View available seats
- Supports First Class and Economy Class
- Menu-driven user input

## Tech Stack
- Java

## Flowchart
![Flowchart](flowchart.png)

## How to Run
1. Open in any Java IDE (VS Code).
2. Compile and run:
   - javac AirlineReservationSystem.java
   - java AirlineReservationSystem
